LPH - Let'sPlayHelper

Have you ever wanted to monitor your free disk space while recording your YouTube-videos? Are you using a recording software without an implemented timer?

This little tool will solve your problems.

Feature overview

  • Display current recording status - Done
  • Display current recording time - Done
  • Display remaining disk space - Nearly done
  • Auto mute for TeamSpeak (if enabled) - Coming in future versions
  • Auto mute for Skype (if enabled) - Done (using old API)
  • Timer alerts - Done (buggy though)
  • easy update checking
  • many more features added in future versions with constant developing

We need your help

Do you have an idea of a great feature, that would fit in this project - code it for yourself and send us a pull request!
